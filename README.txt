U-OV 99 Verhalen website
https://jessejros.github.io/ProjectWeb/

Bus vervoer bedrijf U-OV bied haar reizigers een verhalen website aan waar bezoekers die met de bus reizen leuke verhalen op kunnen lezen. Deze verhalen zijn vormgegeven en geheel interactief.

De code is geschreven voor de website waar deze verhalen worden getoond. De primaire doelen van de lezers zijn dat zij zo snel mogelijk een leuk verhaal willen vinden. Zij willen verhalen kunnen opslaan om ze zo snel terug te vinden en verhalen filteren op genre.

Dit is allemaal mogelijk op de website en in de code staat boven elk onderdeel van de website wat het is.
